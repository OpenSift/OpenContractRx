# Code of Conduct

We follow the Contributor Covenant spirit: be respectful, inclusive, and constructive.

Unacceptable behavior includes harassment, discrimination, threats, doxxing, or sustained disruption.

## Enforcement
Project maintainers may warn, restrict, or remove contributions that violate this code.

## Reporting
If you need to report an issue, open a private security report if it involves sensitive info, otherwise open a normal issue.
